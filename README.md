# Mental-Health-Awareness-Site
This is a simple website built using HTML, CSS & JavaScript. The site is used to spread awareness about mental health and reduce the stigma surrounding mental illness and mental health conditions.

## Features
* Clean and modern design inspired by the android application, Hope
* Fully responsive layout optimized for all devices
* User-friendly interface for easy navigation
* Stylish hover effects added for visual appeal
